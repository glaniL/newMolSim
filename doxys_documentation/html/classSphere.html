Detailed Description

This is a class representing a 3D sphere of particles.

One sphere can save a 3D set of particles, which are located next to each other. There is no isolated particle in one sphere.

std::vector<utils::Vector<double, 3> > Sphere::listOfCenters
private

A list of centers of parallel circles along the Oz axis. The distance between 2 nearest circles is meshWidth. There are (2*radius + 1) elements in this list. Is private and needed for building the sphere (Alternative 2: Bresenham).

std::vector<int> Sphere::listOfRadii
private

A list of radii of parallel circles along the Oz axis, whose centers are stored in listOfCenters. Both listOfCenters and listOfRadii synchronize with eachother. For example: The circle with listOfCenters[i] has radius listOfRadii[i]. There are (2*radius + 1) elements in this list. Is private and needed for building the sphere (Alternative 2: Bresenham).
